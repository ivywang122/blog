JS Notes: Closure 閉包 & Callback 回呼

了解閉包

參考 MDN 文件：Closure

閉包(Closure) 是函數的組合，以及該宣告函數所包含的作用域環境(lexical environment)

Closure 範例

function makeAdder(x) {
  return function(y) {
    return x + y;
  };
}

var add5 = makeAdder(5);
var add10 = makeAdder(10);

console.log(add5(2));  // 7
console.log(add10(2)); // 12

迴圈中常見的錯誤

function buildFuct(){
    var arr = [];
    for(var i = 0; i < 3; i++) {
        arr.push(
            function(){
                console.log(i);
            }
        )
    }
}

var fs = buildFuct();
fs[0]();    // 3
fs[1]();    // 3
fs[2]();    // 3

解決方式

利用 IIFE (Immediately Invoked Function Expression)， 參考 MDN：IIFE

function buildFuct2(){
    var arr = [];
    for(var i = 0; i < 3; i++) {
        arr.push(
            (function(j) {
                return function() {
                    console.log(j)
                }
            })(i)
        )
    }
}

var fs = buildFuct2();
fs[0]();    // 0
fs[1]();    // 1
fs[2]();    // 2

不使用更多 Closures 的情況下運行：ES6 的 let

function buildFuct3(){
    var arr = [];
    for(var i = 0; i < 3; i++) {
        let j = i;
        arr.push(
            function(){
                console.log(j);
            }
        )
    }
}

var fs = buildFuct3();
fs[0]();    // 0
fs[1]();    // 1
fs[2]();    // 2

Callback 函數

呼叫一個 函數 A，傳入 函數 B，當 A 函數執行結束後，回呼執行傳入的函數 B

function callbackFuct(callback) {
    // some code to work...
    callback();
}

callbackFuct(function() {
    console.log('callback!');
})
