前言

本篇為夜間去台大修 C# 整理的初學者筆記。目前理解 Visual Studio 為一個 IDE (整合開發環境)，可以開發跨裝置的應用程式。而 C# 為物件導向的程式語言，可以建置各種在 .NET Framework 上執行的應用程式。更多詳細內容 Microsoft 已有提供文件教學：C# 語言和 .NET Framework 簡介

若之後有新的心得與筆記內容，會繼續編輯本篇文章。

型態轉換

數字間轉換

  • 明確轉型：(儲存空間)大 -> 小
  • 隱含轉換：(儲存空間)小 -> 大

明確轉型

語法：型態 轉換後變數名 = (型態)原變數名

double a = 3.1415926;
float b = (float)a;

隱含轉換

語法：型態 轉換後變數名 = 原變數名

float a = 3.1415926f;
double b = a;   // 小 -> 大 直接轉，不需指定型態
double c = (double)a; // 也可以

數字轉字串

語法：string str = num.ToString();

int num = 10;
string str = num.ToString();

字串轉數字

Parse

若字串為合法的數字，可以使用；若不合法，執行程式會發生錯誤。

語法：數字型態 num = 數字型態.Parse(str);

// 字串需為合法數字
string str = "3.14";
float pi = float.Parse(str);

TryParse

防止程式執行錯物的另一種字串轉數字的方法。

語法：數字型態.TryParse(str, out num);

int num;
string str = "50";
// 如果轉換成功，num 會被覆寫為 50；轉換失敗則為 0
// 執行後的返回值 為 Boolean
bool result = int.TryParse(str, out num);

陣列

陣列宣告

語法 1：型別[] 變數名稱 = new 型別[長度]
語法 2：型別[] 變數名稱 = new 型別[]{val1, val2,…}

// 1
int[] numbers = new int[4];

// 2
int[] numbers2 = new int[] { 1, 2, 3, 4 };

取得陣列長度

int index = numbers.Length;

其他資料型別

Nullable Type

int? num = null;

Dynamic Type

一般來說不建議使用，此型別類似 JavaScript 可以動態強制轉型。

dynamic a = 5;
dynamic b = 3;

// 整數相加
Console.Write(a+b); // 8

// 可直接轉型
a = "5";
Console.Write(a+b); // 53 當成字串相連
