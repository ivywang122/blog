Visual Studio 管理專案

2018-05-03
Visual Studio

Contents

  • 前言
    • 建立方案

前言

專案 是當我們要建立 app、web 或應用程式等，需要開始的一個包含所有程式碼、圖片影像、綜合所有服務或元件等項目的一個集合。

那有一個問題是如果今天多人協作一個大型專案的各項功能，每個人都有一個專案的時候，這些子集合要整理的時候要怎麼辦？Visual Studio 提供一個特別的 方案 (solution) 功能，可以建立關聯各個專案的特殊組件檔案 (副檔名 .sln) 來管理所有專案。

更多詳細說明可參考：Visual Studio 中的方案和專案
