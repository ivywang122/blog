前言

Hugo 是一個由Golang實現的靜態網站生成器(Static Site Generator, SSG)，不像WordPress, Ghost 和 Drupal 要花較多時間建立頁面，還有後續動態頁面更新的問題及安全性等隱憂。因為不是每個網站都需要 Server 來動態生成內容，和數據庫等，因此想架設一個筆記部落格，SSG是一個) 而靜態網頁的優點在於：

快速部署，維護簡單，可拓展性高，高效能返回已渲染好的網頁

而 Jekyll (另一個靜態網頁生成器先驅) 是以 RubyGems 環境，對Windows的使用者非常不友善QQ，在安裝前置作業很花時間，之前試著玩看看覺得青春歲月就這樣去了…

所以我直接放棄 Jekyll 選擇較好入門的 Hugo 和 Github Pages 搭建自己的 Blog。


Windows 安裝

Hugo 官網有提供教學影片可參考：Install Hugo | Hugo

建立資料夾

  1. 建立 C:\Hugo\bin 作為儲存執行文件，也可以放在其他地方
  2. 建立 C : \Hugo\Projects\blog 作為新專案目錄

下載 Hugo

  1. 下載最新 version 的壓縮文件 Releases · gohugoio/hugo · GitHub
  2. 解壓縮內容至剛剛建立的 C:\Hugo\bin
  3. 若 hugo 執行檔將被命名為 hugo_hugo-version_platform_arch.exe，請將文件重新命名為 hugo.exe 使用

將 Hugo 添加到的 Windows 環境變數設置中

  1. 開始［搜尋］檢視進階系統設定 > 進階 > 環境變數 > 使用者變數 > Path 編輯
  2. 新增路徑 C:\Hugo\bin 確定

下載 Hugo

  1. Releases · gohugoio/hugo · GitHub 下載
  2. 解壓縮內容至剛剛建立的 C : \Hugo\bin
    如果壓縮檔案中 hugo 未正確顯示副檔名，請重新命名為：hugo.exe
