Visual Studio 管理專案

2018-05-03
Visual Studio

前言

專案 是當我們要建立 app、web 或應用程式等，需要開始的一個包含所有程式碼、圖片影像、綜合所有服務或元件等項目的一個集合。

那有一個問題是如果今天多人協作一個大型專案的各項功能，每個人都有一個專案的時候，這些子集合要整理的時候要怎麼辦？Visual Studio 提供一個特別的 方案 (solution) 功能，可以建立關聯各個專案的特殊組件檔案 (副檔名 .sln) 來管理所有專案。

JS Notes: 嚴格模式

2018-05-02
JavaScript

前言

嚴格模式對 JavaScript 語意上做了一些修正：

  • 透過拋出錯誤的方式，消除 JavaScript 默認上的錯誤
  • 修正會阻礙 JavaScript 引擎進行最佳化的錯誤，相同的程式碼使用嚴格模式可以比非嚴格模式運作更快
  • 禁止使用一些有可能會被未來版本 ECMAScript 定義的語法

2018-04-18
JavaScript

繼承

一個物件取用另一個物件的屬性或是方法，回歸本質來思考，繼承的目的就是在擴充原有物件定義不足的地方。不同程式語言的物件導向特性可分為：

  • 古典繼承 (Classical Inheritance)
  • 原型繼承 (Prototypal Inheritance)

而在 JavaScript 中使用的是原型繼承 (Prototypal Inheritance)。
